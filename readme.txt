
jupyter install instructions:

Windows:
In PyCharm:
1. Right click on setup_jupyter_win.bat
2. Select "Open In", then "Explorer"
3. Double click on setup_jupyter_win.bat

Mac:
In PyCharm:
1. Right click on setup_jupyter_mac.sh
2. Select "Open In", then "Terminal"
3. At the command prompt, type chmod 755 setup_jupyter_mac.sh
4. Type "./setup_jupyter_mac.sh" (do NOT type the quotation marks, but do type everything else, including the period and slash at beginning)
